Class: Client

Client(api)

Class representing a Client.

A Client is a software that acts like an agent between a user and a server whose resources the user wants to access. By establishing a connection with a server, a client can request resources exposed by the server through endpoints, each one serving a distinct logical section of the resources.
In order to establish a connection with the server, a client must first be granted an authorization by the server. Supported authentication methods are:

  • OAuth 2.0 Bearer Token
